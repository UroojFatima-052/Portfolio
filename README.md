# Urooj Fatima — Personal Portfolio

A personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no build tools — just clean, fast static files.

## Live Site

Deployed on Netlify: *(add your URL here after deployment)*

## Sections

- **Hero** — Animated intro with typewriter effect and stat counters
- **About** — Bio, highlight cards, and quick stats
- **Skills** — Animated progress bars + skill icon grid
- **Services** — Web Development, AI/ML, Data Science, Python Development
- **Projects** — Zariya, Sentiment Analysis, Customer Segmentation, Energy Forecasting
- **Contact** — Contact form with validation + social links

## Tech Stack

| Layer | Tech |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, glassmorphism, animations) |
| Logic | Vanilla JavaScript |
| Icons | Font Awesome 6 |
| Fonts | Poppins, Inter, Montserrat (Google Fonts) |
| Hosting | Netlify |

## Features

- Dark / Light theme toggle
- Scroll-reveal animations (re-trigger on every scroll)
- Typewriter effect cycling through roles
- Animated skill progress bars
- Project filter buttons (All / Full-Stack / AI-ML / Data Science)
- Contact form with client-side validation
- Fully responsive (mobile, tablet, desktop)
- Loader with letter-drop animation
- Scroll to top button
